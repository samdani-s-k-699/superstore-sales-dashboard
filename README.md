Superstore Sales Dashboard (Excel Project)

This project focuses on transforming raw sales data into an interactive and insightful dashboard using Microsoft Excel. The goal is to analyze business performance and present key metrics in a clear and visually appealing way.

🔹 Project Overview

Cleaned and structured raw dataset into a usable format
Built an interactive Excel dashboard using Pivot Tables and Charts
Created key performance indicators (KPIs) to track sales performance

🔹 Key Features

📈 Dynamic dashboard with slicers (Region, Category)

📊 KPIs including Total Sales, Total Profit, and Profit Margin

🔍 Interactive filtering to explore data by different dimensions

📉 Visual insights through bar charts, line charts, and category comparisons

🔹 Key Insights

💰 Total Sales: ~2.1M

📊 Profit Margin: ~12%

🏆 Technology category generated highest sales

🌍 West region showed strong performance

⚠️ High discounts impacted profitability in some cases
